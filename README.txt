Machine config

Prerequisites:
nix
brew (macOS)

Install:
Run `make HOST=<host>`

On Linux you can run GNUMake by adding it to the environment
`nix-shell -p gnumake`

Post-install:
- Configure macOS display settings

- Set up macOS keyboard shortcuts
  - Tiling via app shortcuts (Left, Right, Fill, Centre, Center)
  - Mission control space switching
  - New shell tab at folder via services

- Configure Finder sidebar

- Configure Firefox toolbars

- Add GitHub SSH keys
